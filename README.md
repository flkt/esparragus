# esparragus

generador de sistemas de diseño 

## uso

### descarga

```sh
git clone git@github.com:flkt/esparragus.git
```

### instala

```sh
cd esparragus
npm install
```

### levanta la documentacion

```sh
npm run dev
```

### cambia de rama para que no sobreescribas el original

```sh
git checkout -b otro-proyecto
```
